    - ユーザーが見やすいように1ループごとにクリアでコンソールの文字を削除

    - IDから対象の内容を閲覧できるように
- シーケンス図の工夫
    - なるべく簡易かつ、実際の処理の流れに沿うように書き、綺麗にまとまるように書いた。
